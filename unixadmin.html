thomasTrenholme/unixadmin

UNIX/Linux System Administration

During Summer 2022, I had the pleasure of taking CS 470 Unix System Administration with Professor Bartoli at San Diego State University. During the 12-week course, I set up a private business network consisting of a DNS and openLDAP hosting server via openbsd, a mail server via freebsd, an NFS file hosting server via almalinux, a docker containerized web server via Ubuntu that hosted an nginx server and portainer dashboard, an AIX server via IBM, a solaris server with openIndiana, a backups hosting server with ESXI, and a gitlab server with microsoft Azure. Configuring the network to use a secured SSH connection between virtual machines, and setting up strong security measures for the network was a point of emphasis.

Creating all of these services and servers was a fufilling experience that has allowed me to gain a great amount of knowledge surrounding the Unix/Linux system administration field. Alongside with taking Operating Systems in Spring 2022 at San Diego State University, I have found myself a great deal of pleasure working within Linux and being a system administrator.
