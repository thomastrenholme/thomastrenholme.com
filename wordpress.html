WordPress Plugin development

    • In December of 2021, Santa Cruz Savory, a company of passionate entrepreneurs including Sean Moran, Josh Zueck, Kristian Windsor, and myself launched. We have launched a wordpress plugin publically available at santacruzsavory.com. This plugin allows recipe website owners to conveniently allow their readers to purchase the ingredients through popular grocery delivery services. Recently in beta testing, we are happy to announce that the plugin has officially released!

    My partner on the development team of the plugin, Kristian has developed a responsive API that logs recipe data and quickly parses recipe information into a JSON format. This JSON is then sent to the plugin's backend to be parsed and stored in the database. The plugin is able to use grocery services APIs to speedily deliver a working shopping cart link to the user. I have developed the front end of the plugin using jQuery, PHP, and CSS to create a responsive feeling button that does not intrude on our users' websites nor privacy. The plugin is fully functional and supports many different types of WordPress recipe websites, and a chrome extension is planned for the future.
